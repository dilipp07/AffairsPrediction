About the dataset

Variable name definitions:

rate_marriage : How rate marriage, 1 = very poor, 2 = poor, 3 = fair, 4 = good, 5 = very good

yrs_married : No. years married. Interval approximations. See original paper for detailed explanation.

children : No. children

religious : How relgious, 1 = not, 2 = mildly, 3 = fairly, 4 = strongly

educ : Level of education, 9 = grade school, 12 = high school, 14 = some college, 16 = college graduate, 17 = some graduate school, 20 = advanced degree

occupation : occ_2 = farming, agriculture; semi-skilled, or unskilled worker; occ_3 = white-colloar; occ_4 = teacher counselor social worker, nurse; artist, writers; technician, skilled worker, occ_5 = managerial, administrative, business, occ_6 = professional with advanced degree

SELECT 1 FOR YOUR OCCUPATION ELSE SELECT 0

occupation_husb : Husband's occupation. Same as occupation.
